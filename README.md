# 💍 I•Match Dating Bot v2.0

Телеграм-бот и Mini App для анкетирования в международном дейтинг-агентстве.

![Version](https://img.shields.io/badge/version-2.0-pink)
![License](https://img.shields.io/badge/license-MIT-blue)

## ✨ Возможности

### Telegram Bot
- 🌍 **Мультиязычность** — Русский и Английский
- 📱 **Два способа заполнения** — Mini App или классический бот
- 💾 **База данных SQLite** — надёжное хранение анкет
- 📸 **Загрузка нескольких фото** — до 10 штук
- ✅ **Валидация** — email, ссылки на соцсети, возраст 18+
- 🔐 **Скрытая админ-панель** — только для администраторов
- 📊 **Статистика** — количество анкет, статусы

### Mini App
- 🎨 **Современный дизайн** — градиенты, анимации, стеклянные эффекты
- 🌓 **Тёмная и светлая тема** — автоопределение из Telegram или ручной выбор
- 🌍 **Мультиязычность** — полная поддержка RU/EN
- 💾 **Автосохранение** — данные не потеряются при закрытии
- 📱 **Адаптивность** — работает на любых устройствах
- 📊 **Прогресс-бар** — наглядное отображение заполненности
- 🎯 **Навигация по разделам** — удобное перемещение между разделами
- 📸 **Превью фотографий** — просмотр и удаление загруженных фото

## 🛠 Админ-команды

Команды доступны только администраторам и не видны обычным пользователям.

| Команда | Описание |
|---------|----------|
| `/admin` | Открыть админ-панель со статистикой |
| `/addadmin USER_ID @username` | Добавить нового администратора |
| `/removeadmin USER_ID` | Удалить администратора |
| `/profiles` | Показать последние 10 анкет |
| `/approve ID` | Одобрить анкету |
| `/reject ID` | Отклонить анкету |

## 👤 Пользовательские команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу с ботом |
| `/restart` | Перезапустить анкету |

## 📱 Mini App

Mini App — это React-приложение с красивым интерфейсом для заполнения анкеты.

### Деплой на Vercel

1. Соберите проект: `npm run build`
2. Загрузите папку `dist` на Vercel
3. Обновите `WEB_APP_URL` в боте

### Локальная разработка

```bash
npm install
npm run dev
```

## 📊 База данных

Бот использует SQLite. База создаётся автоматически при первом запуске.

### Таблицы

- **admins** — список администраторов
- **profiles** — анкеты пользователей
- **photos** — фотографии к анкетам

### Просмотр базы

```bash
sqlite3 imatch_database.db
.tables
SELECT * FROM profiles LIMIT 5;
```
